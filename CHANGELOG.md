# Changelog

### Changed

- Changed supported Minecraft version to 1.21.3.

## [1.1.2+1.21.2]

### Changed

- Changed supported Minecraft version to 1.21.2.

- Added additional options for specific control over the chance of flatulence.
- Added flatulence to players.
